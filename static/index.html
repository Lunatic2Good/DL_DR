About This Project

Comprehensive information about the project, models, and dataset

Project Overview

This is a Diabetic Retinopathy (DR) Classification System that uses multiple deep learning models to analyze retinal images and predict whether an eye shows signs of diabetic retinopathy or is healthy.

The system compares predictions from different model architectures to provide comprehensive analysis and performance metrics.

Models Used

12 Layer CNN

Custom 12-layer Convolutional Neural Network designed for retinal image classification. Features multiple convolutional and pooling layers for feature extraction.

  • Parameters: 2.3M
  • Accuracy: 100%
  • Architecture: Custom CNN with batch normalization
ResNet50

Deep Residual Network with 50 layers, leveraging skip connections to enable training of very deep networks without degradation.

  • Parameters: 25.6M
  • Accuracy: 85%
  • Architecture: Residual blocks with identity mapping
5 Layer CNN

Lightweight 5-layer Convolutional Neural Network optimized for fast inference while maintaining good classification performance.

  • Parameters: 1.1M
  • Accuracy: 89%
  • Architecture: Compact CNN design
BEiT (HuggingFace)

Bidirectional Encoder representation from Image Transformers - a vision transformer model pre-trained on ImageNet and fine-tuned for medical image classification.

  • Parameters: 86.5M
  • Accuracy: 90-99%
  • Architecture: Vision Transformer (ViT-based)
Swin Transformer

Shifted Window Transformer - a hierarchical vision transformer that uses shifted windows for efficient computation and better feature representation.

  • Parameters: 88.0M
  • Accuracy: 90-99%
  • Architecture: Hierarchical Swin Transformer

Dataset

The models were trained on a retinal fundus image dataset for diabetic retinopathy classification.

Image Type: Retinal Fundus Images
Classes: 2 (Healthy / DR)
Image Size: 224x224 pixels (preprocessed)
Preprocessing: Resize, Normalization (ImageNet/Simple)
